# 专跨股对冲分析终端 (Stock Hedging Terminal)

本项目是一个为投资专业人士设计的实时股票价差监控与对冲分析工具。通过现代金融美学界面，提供分时级实时对比与长达 30 天的历史收盘对冲分析。

## ✨ 核心特性

- **现代金融视觉 (Modern UI)**：采用 Slate-50 极简浅色系风格，针对长时间盯盘优化的非刺眼配色与微卡片设计。
- **双维度对冲分析 (Dual-View)**：
    - **分时价差**：实时追踪盘中每分钟的差值跳变。
    - **历史价差**：直连腾讯官方 **fqkline (前复权K线)** 引擎，获取近 30 个交易日的真实收盘价序列。
- **ISO-GRID 数据对齐**：采用日期键值映射（Date-Key Join）算法，强制对冲双股在同一自然日下运算，完美解决停牌、数据断档导致的时间轴漂移。
- **高精度交互体验**：
    - **1元整数刻度**：纵坐标锁定 1 元一个阶梯，大幅提升对比可读性。
    - **智能 Tooltip**：悬停即显股票中文名与具体收盘价。
- **工业级反爬加固**：
    - 全仿真浏览器 Headers 注入（User-Agent, Referer）。
    - 后端 10s 智能缓存机制。
    - 历史数据请求间 0.3s 防限流延迟。
    - 彻底绕过 Windows 系统损坏代理的脱壳直连方案。

## 🛠️ 技术栈

- **前端**: React 19 + Vite 6 + Tailwind CSS 4 + Recharts (金融面积图)
- **后端**: Django 4.2 + Requests (Session级脱壳请求) + SQLite
- **数据源**: 腾讯财经实时行情与官方历史复权日线接口

## 🚀 快速启动

本项目支持 Windows 一键环境自动化处理：

1. **一键运行**: 直接双击根目录下的 `start.bat`。
2. **自动化流程**:
   - 自动同步 Python 依赖 (`requirements.txt`)。
   - 自动编译前端生产包 (`npm run build`)并托管至静态资源目录。
   - 自动启动服务并在默认浏览器打开：`http://localhost:3000`。

## ⚙️ 使用配置

- **添加/删除股票**: 点击右上角“齿轮”图标打开侧边配置面板。
- **视图切换**: 在主界面上方点击“分时价差”或“历史价差”进行无缝切换。
- **持久化**: 监控队列自动保存至本地 `localStorage`，刷新不丢失。
